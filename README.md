# Breakpoint Mobile

React Native mobile app for the Breakpoint Podcast, built with Expo and TypeScript.

## Features

- Home screen with podcast positioning, featured episodes, and quick actions
- Episodes tab with search, category chips, and rich episode cards
- Episode detail screen with show notes, links, and share/copy actions
- About screen with mission, listening platforms, and contact details
- Subscribe screen with email capture UI and platform shortcuts
- Reusable themed components and centralized podcast content data

## Getting started

```bash
npm install
npm run start
```

Then open the app in Expo Go, an iOS simulator, or an Android emulator.

## Validation

```bash
npm run typecheck
```
